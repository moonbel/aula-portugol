OPERADORES EM JAVASCRIPT

Aprenda de forma completa, detalhada e visualmente bonita o que são os operadores em JavaScript, como eles funcionam e veja exemplos práticos funcionando em tempo real.

O que são Operadores?

Os operadores em JavaScript são símbolos especiais usados para realizar operações em valores e variáveis.

Eles são fundamentais em qualquer programa porque permitem:

  • Fazer cálculos matemáticos
  • Comparar valores
  • Criar condições
  • Tomar decisões
  • Manipular dados

Exemplo simples:

let soma = 10 + 5;

Nesse caso:

  • 10 e 5 são operandos
  • + é o operador

1. Operadores Aritméticos

São usados para realizar cálculos matemáticos.

Principais operadores:

  • + → Soma
  • - → Subtração
  • * → Multiplicação
  • / → Divisão
  • % → Resto da divisão
  • ** → Potência
let a = 10;
let b = 5;

console.log(a + b); // 15
console.log(a - b); // 5
console.log(a * b); // 50
console.log(a / b); // 2
console.log(a % b); // 0

2. Operadores de Comparação

São usados para comparar valores. O resultado sempre será:

  • true
  • false

Principais operadores:

  • == → Igual
  • === → Estritamente igual
  • != → Diferente
  • > → Maior que
  • < → Menor que
  • >= → Maior ou igual
  • <= → Menor ou igual
console.log(10 > 5); // true
console.log(10 < 5); // false
console.log(10 == "10"); // true
console.log(10 === "10"); // false

3. Operadores Lógicos

Servem para combinar condições.

  • && → E lógico
  • || → OU lógico
  • ! → NÃO lógico
let idade = 20;
let temCarteira = true;

console.log(idade >= 18 && temCarteira);
// true

4. Operadores de Atribuição

Usados para atribuir valores às variáveis.

  • = → Atribuição simples
  • += → Soma e atribui
  • -= → Subtrai e atribui
  • *= → Multiplica e atribui
let numero = 10;

numero += 5;

console.log(numero);
// 15

5. Incremento e Decremento

  • ++ → Incrementa 1
  • -- → Decrementa 1
let x = 5;

x++;

console.log(x);
// 6
